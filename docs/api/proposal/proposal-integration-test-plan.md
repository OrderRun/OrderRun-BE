# Proposal Integration Test Legacy Reference

이 문서는 과거 Proposal 통합 테스트 체크리스트 위치를 유지하기 위한 참조 문서다.

현재 테스트 보장 범위는 [`../../domains/proposal/test-scenarios.md`](../../domains/proposal/test-scenarios.md)를 우선한다.
테스트 파일/함수 파생 색인은 [`../../generated/test-inventory.md`](../../generated/test-inventory.md)를 따른다.

새 테스트 시나리오를 추가하거나 변경할 때는 이 파일이 아니라 도메인 테스트 보장 문서를 갱신한다.
