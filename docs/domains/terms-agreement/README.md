# Terms Agreement Domain

Terms Agreement는 사용자의 약관 동의 상태를 사용자 식별자에 종속해 관리한다.

## 책임

- 필수/선택 약관 동의 상태를 생성하거나 갱신한다.
- 인증된 사용자만 본인의 약관 상태를 변경할 수 있게 한다.
- 요청 validation과 모델 계약을 API 문서와 맞춘다.
