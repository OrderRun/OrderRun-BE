# Proposal 게시글 신고

## 사용자 흐름

사용자는 공개 모집 중인 다른 사용자의 Proposal을 신고할 수 있다. 신고 화면은 활성 신고 사유를 조회하여 다음 다섯 항목을 표시한다.

1. 광고 또는 스팸이에요
2. 욕설이나 혐오 표현이 있어요
3. 거짓 정보 같아요
4. 부적절한 사진이나 내용이에요
5. 기타

상세 사유는 선택이며, 입력하면 500자 이내여야 한다. 동일 사용자는 같은 게시글을 한 번만 신고할 수 있고 작성자는 자신의 게시글을 신고할 수 없다.

## 관리자 처리

신고는 `PENDING`으로 접수된다. 관리자가 승인하면 신고는 `ACCEPTED`가 되고 Proposal은 `REPORTED`로 전환된다. 이 게시글은 공개 목록과 신규 Offer 대상에서 제외되며, 기존 대기 Offer는 거절된다. 반려하면 신고만 `REJECTED`로 바뀌고 Proposal은 유지된다.

상세 API 계약은 [`../api-spec/README.md`](../api-spec/README.md), 상태 정책은 [`../domain.md`](../domain.md)를 따른다.
