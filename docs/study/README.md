# Study Notes

이 디렉터리는 학습, 실험, 개인/팀 메모를 보관한다.

## 책임

- Python, SQLAlchemy, 도구 사용법 같은 학습 메모
- 공식 결정 전의 실험 기록
- 온보딩이나 개념 이해를 돕는 참고 자료

## 금지

- API 계약, 도메인 상태, 데이터 모델의 정본으로 사용하지 않는다.
- 설계 결정의 최종 근거로 직접 참조하지 않는다.
- 실행 계획이나 완료 기준을 이 디렉터리에 두지 않는다.
